컬러 가이드

TRT 브랜드 아이덴티티 컬러의 Deep Green(#004D45)을 Core Color(Primary Color)로 지정합니다.
명암비 기반으로 생성된 색상을 사용하여 다양한 상황에 사용하는 색상 시스템을 만들어 사용합니다. 컬러별 단계 10단계(때로는 11단계 / 25-900)을 지원하고 디자인 색상 토큰으로 사용합니다.

Semantic Color

Text

타이틀, 본문, 주석 등의 텍스트는 Emphasis Level에 따라 Highest, Mid, Low, Lowest 로 정의합니다.
버튼이나 강조하는 텍스트를 사용해야 하는 경우 Primary 컬러를 사용합니다.
에러, 경고 등의 텍스트에서는 Red 컬러를 사용하고 텍스트 링크중 활성화 표시를 해야하는 경우 Blue 컬러를 사용합니다.

  • Highest Gray 90 #1A1A1A

  • Medium --Color-Gray-70 #4D4D4D

  • Low Gray 60 #666666

  • Lowest Gray 50 #808080

  • White White #FFFFFF

  • Primary Primary 800 #004D45

  • Nagative Red 400 #FA4156

  • Link Blue 600 #006BB2

Table

Table의 Top/Bottom Line과 구분선에 사용합니다.

  • Topline Gray 90 #1a1a1a

  • Line Gray 10 #e5e5e5

  • Title_bg Primary 25 #F2F7F7

Border

Secondary Button, Checkbox, Radio Button 등은 시인성을 높이기 위해 Border-Secondary를 적용합니다.
Selectbox, Inputbox, Card, List 등의 경우엔 Border-Tertiary를 적용합니다.
